
# [21. 合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/)

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        if not list1:
            return list2
        if not list2:
            return list1
        if list1.val <= list2.val:
            head = list1
            list1 = list1.next
        else:
            head = list2 
            list2 = list2.next
        cur = head
        while list1 and list2:
            if list1.val <= list2.val:
                cur.next = list1
                list1 = list1.next
                cur = cur.next
            else:
                cur.next = list2
                list2 = list2.next
                cur = cur.next

        if list1:
            cur.next = list1
        elif list2:
            cur.next = list2
        return head
```


# [86. 分隔链表](https://leetcode.cn/problems/partition-list/)

弄两个head，一个链小于的，一个链接大于的，连在一起就好了
记得要断开每个链表的链接。没断开会无限循环（最后一个）



识别了链表节点指针断开的必要性。
因为节点被接入新链表时，它的 next 还指向原链表的下一个节点。

举例说明，假设原链表是 1 -> 3 -> 2，x=2：

p=1，接入 dummy1 链：dummy1 -> 1 -> 3（旧next）
p=3，接入 dummy2 链：dummy2 -> 3 -> 2（旧next）
p=2，接入 dummy1 链：dummy1 -> 1 -> 2
最后执行 p1.next = dummy2.next，结果变成：

dummy1 -> 1 -> 2 -> 3 -> 2 -> 3 -> ...（无限循环！）
因为 3 的 next 还指着 2，2 的 next 最终又指回了 3。

断开的作用就是清空每个节点的旧 next，防止拼接后形成环。

```python
class Solution:
    def partition(self, head: ListNode, x: int) -> ListNode:
        # 存放小于 x 的链表的虚拟头结点
        dummy1 = ListNode(-1)
        # 存放大于等于 x 的链表的虚拟头结点
        dummy2 = ListNode(-1)
        # p1, p2 指针负责生成结果链表
        p1, p2 = dummy1, dummy2
        # p 负责遍历原链表，类似合并两个有序链表的逻辑
        # 这里是将一个链表分解成两个链表
        p = head
        while p:
            if p.val >= x:
                p2.next = p
                p2 = p2.next
            else:
                p1.next = p
                p1 = p1.next
            # 不能直接让 p 指针前进，
            # p = p.next
            # 断开原链表中的每个节点的 next 指针
            temp = p.next
            p.next = None
            p = temp
        # 连接两个链表
        p1.next = dummy2.next

        return dummy1.next
```

[23. 合并 K 个升序链表](https://leetcode.cn/problems/merge-k-sorted-lists/)

```python
ListNode.__lt__ = lambda a, b: a.val < b.val  # 让堆可以比较节点大小

class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        cur = dummy = ListNode()  # 哨兵节点，作为合并后链表头节点的前一个节点
        h = [head for head in lists if head]  # 把所有非空链表的头节点入堆
        heapify(h)  # 堆化
        while h:  # 循环直到堆为空
            node = heappop(h)  # 剩余节点中的最小节点
            if node.next:  # 下一个节点不为空
                heappush(h, node.next)  # 下一个节点有可能是最小节点，入堆
            cur.next = node  # 把 node 添加到新链表的末尾
            cur = cur.next  # 准备合并下一个节点
        return dummy.next  # 哨兵节点的下一个节点就是新链表的头节点
```
