
# [21. 合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists/)

```python
class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        if not list1:
            return list2
        if not list2:
            return list1
        if list1.val <= list2.val:
            head = list1
            list1 = list1.next
        else:
            head = list2 
            list2 = list2.next
        cur = head
        while list1 and list2:
            if list1.val <= list2.val:
                cur.next = list1
                list1 = list1.next
                cur = cur.next
            else:
                cur.next = list2
                list2 = list2.next
                cur = cur.next

        if list1:
            cur.next = list1
        elif list2:
            cur.next = list2
        return head
```


# [86. 分隔链表](https://leetcode.cn/problems/partition-list/)

弄两个head，一个链小于的，一个链接大于的，连在一起就好了
记得要断开每个链表的链接。没断开会无限循环（最后一个）



识别了链表节点指针断开的必要性。
因为节点被接入新链表时，它的 next 还指向原链表的下一个节点。

举例说明，假设原链表是 1 -> 3 -> 2，x=2：

p=1，接入 dummy1 链：dummy1 -> 1 -> 3（旧next）
p=3，接入 dummy2 链：dummy2 -> 3 -> 2（旧next）
p=2，接入 dummy1 链：dummy1 -> 1 -> 2
最后执行 p1.next = dummy2.next，结果变成：

dummy1 -> 1 -> 2 -> 3 -> 2 -> 3 -> ...（无限循环！）
因为 3 的 next 还指着 2，2 的 next 最终又指回了 3。

断开的作用就是清空每个节点的旧 next，防止拼接后形成环。

```python
class Solution:
    def partition(self, head: ListNode, x: int) -> ListNode:
        # 存放小于 x 的链表的虚拟头结点
        dummy1 = ListNode(-1)
        # 存放大于等于 x 的链表的虚拟头结点
        dummy2 = ListNode(-1)
        # p1, p2 指针负责生成结果链表
        p1, p2 = dummy1, dummy2
        # p 负责遍历原链表，类似合并两个有序链表的逻辑
        # 这里是将一个链表分解成两个链表
        p = head
        while p:
            if p.val >= x:
                p2.next = p
                p2 = p2.next
            else:
                p1.next = p
                p1 = p1.next
            # 不能直接让 p 指针前进，
            # p = p.next
            # 断开原链表中的每个节点的 next 指针
            temp = p.next
            p.next = None
            p = temp
        # 连接两个链表
        p1.next = dummy2.next

        return dummy1.next
```

```python
import heapq

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

    # 重载比较运算符，方便将 ListNode 加入最小堆
    def __lt__(self, other):
        return self.val < other.val

class Solution:
    def mergeKLists(self, lists):
        if not lists:
            return None
        # 虚拟头结点
        dummy = ListNode(-1)
        p = dummy
        # 优先级队列，最小堆
        pq = []
        # 将 k 个链表的头结点加入最小堆
        for i, head in enumerate(lists):
            if head is not None:
                heapq.heappush(pq, (head.val, i, head))

        while pq:
            # 获取最小节点，接到结果链表中
            val, i, node = heapq.heappop(pq)
            p.next = node
            if node.next is not None:
                heapq.heappush(pq, (node.next.val, i, node.next))
            # p 指针不断前进
            p = p.next
            
        return dummy.next
```
